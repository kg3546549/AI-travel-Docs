## 여행 기본 조건 정의

케이스 분기의 핵심. 각 조건이 "확정/미정"이냐에 따라 AI가 채워야 할 항목이 결정된다.

### 필수 조건

| # | 조건 | 설명 | 미정일 때 AI 처리 |
|---|------|------|-----------------|
| 1 | **목적지** | 국가 / 도시 (복수 가능) | 스타일 기반으로 여행지 추천 |
| 2 | **일정** | 출발일 + 귀국일 → 박수/일수 | 스타일·거리 기반으로 적정 일수 추천 |
| 3 | **인원** | 몇 명, 관계 (혼자/커플/친구/가족) | 기본값 2인으로 처리 |
| 4 | **예산** | 총 예산 or 1인 예산 (항공 포함 여부) | 해당 목적지 평균 여행 비용 제시 |
| 5 | **여행 스타일** | 맛집, 관광지, 액티비티, 쇼핑, 휴양, 문화 (복수 선택) | 일반적인 조합으로 기본 구성 |

### 선택 조건

| # | 조건 | 없을 때 기본값 |
|---|------|-------------|
| 6 | **출발지** | 서울(인천) |
| 7 | **이동 수단** | 대중교통 |
| 8 | **숙소 유형** | 중간 가격대 호텔 |
| 9 | **일정 강도** | 보통 |

### 케이스 분기 로직

```
목적지 확정 + 일정 확정  →  AI: 스타일/예산에 맞는 세부 일정만 구성
목적지 확정 + 일정 미정  →  AI: 적정 일수 추천 후 일정 구성
목적지 미정             →  AI: 스타일/예산 기반으로 목적지부터 추천
```

## UX 플로우

### 1단계 - 조건 입력 (토스 스타일 스텝)

```
① 어디로 여행 가세요?
   → [미정 - AI 추천] or [국가/도시 검색]

② 언제 가세요?
   → [미정 - AI 추천] or [날짜 선택]  ※ 박수 자동 계산

③ 몇 명이서 가세요?
   → [인원 선택] + [관계: 혼자 / 커플 / 친구 / 가족]

④ 예산은 어느 정도예요?
   → [미정] or [직접 입력] or [이 여행지 평균 비용: OO만원]

⑤ 여행 스타일 (복수 선택)
   → 맛집 / 관광지 / 액티비티 / 쇼핑 / 휴양 / 문화체험

⑥ (선택) 더 디테일하게 설정할까요?
   → 출발지 / 이동수단 / 숙소유형 / 일정강도

⑦ 추가로 원하는 게 있나요? (자유 입력)
   → 예: "해리포터 스튜디오 꼭 가고 싶어요"
```

### 2단계 - 계획 선택

```
AI가 3~4개 요약 후보 생성 (1차 호출)
    ↓
키워드 카드 리스트로 표시
  [ 🍜 맛집 집중형 | 예상 80만원 ]
  [ 🏯 관광지 위주 | 예상 75만원 ]
    ↓
하나 선택 → AI 상세 일정 생성 (2차 호출)
    ↓
지도뷰 + 일정 상세 페이지
```

### 3단계 - 일정 검토

```
전체 지도뷰 (모든 포인트 한눈에)
    ↓
[다음] 버튼 클릭
    ↓
줌아웃 → 카메라 이동 → 다음 장소 줌인 (애니메이션)
하단 카드: 장소명 / 예상 체류 시간 / 메모
    ↓
전체 검토 완료 → 저장 / 공유
```

출력 예시:

```
[도쿄 3박 4일 · 2인 · 예산 80만원 · 맛집+관광지]

1일차  시부야 → 하라주쿠 → 신주쿠
2일차  아사쿠사 → 우에노 → 아키하바라
3일차  오다이바 → 긴자
4일차  공항 이동

총 예상 비용: 항공 28만 / 숙소 24만 / 식비 18만 / 기타 10만 = 약 80만원
```
