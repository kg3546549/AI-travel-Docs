## 한 줄 정의

**"조건 몇 가지만 입력하면, AI가 완성된 여행 일정을 만들어준다"**

여행 계획에 드는 수고를 최소화하는 AI 기반 여행 플래너.
계획부터 실행까지 하나의 앱에서 끝낸다.

- **초기 지원 국가:** 한국, 일본
- **핵심 타겟:** 여행 계획을 직접 짜고 싶지만 귀찮은 사람 (계획형 J)

---

## 차별점

| 기존 서비스 | 우리 서비스 |
|-------------|-------------|
| 직접 검색하고 조합해야 함 | 조건 입력만 하면 완성 일정 생성 |
| 결과가 하나뿐 | 3~4개 옵션 중 선택 |
| 지도/맛집/예약 앱이 따로 | 하나의 앱에서 통합 제공 |
| 계획 후 실행이 분리됨 | 계획 → 실행까지 연속 지원 |
| 혼자만 볼 수 있음 | 친구와 공유 + 공동 열람 |

**벤치마크:** 야놀자 (예약) + 구글맵 (루트) + 스카이스캐너 (가격) 를 하나로 합친 서비스

## 수익 모델

| 단계 | 모델 | 내용 |
|------|------|------|
| Phase 1 | 광고 | 구글 애즈 |
| Phase 2 | 건당 과금 | 일정 생성 1회 1,000원 |
| Phase 2 | 제휴 수수료 | 예약 앱 연결 시 수수료 |
| Phase 3 | 예약 대행 수수료 | 전체 예약 대행 시 약 1만원 |

## 개발 로드맵

### Phase 1 - MVP

- [ ] 웹 메인 페이지 (검색 + 인기 여행지 카드)
- [ ] 여행 조건 입력 플로우 (자유 입력 포함)
- [ ] AI 2단계 호출 (후보 3~4개 → 선택 → 상세)
- [ ] 후보 계획 저장 및 선택 UI
- [ ] 지도 전체뷰 + 스텝 검토 뷰 (줌 애니메이션)
- [ ] Google Places API 연동
- [ ] Amadeus API 연동
- [ ] 카카오톡 공유 + 공개 웹 페이지
- [ ] 친구 초대 (공동 열람)
- [ ] 관리자 페이지 (여행지 관리 + 계획 대량 생성)
- [ ] 구글 애즈 세팅

### Phase 2 - 고도화

- [ ] Naver Local Search API 연동
- [ ] 공동 편집
- [ ] 예약 앱 딥링크 연결
- [ ] 일정 생성 유료화 (1,000원/회)
- [ ] 여행 실행 모드 (GPS 기반)
- [ ] Best 여행계획 카탈로그 (데이터 누적 후)

### Phase 3 - 예약 대행

- [ ] Toss Payments 통합 결제
- [ ] 예약 자동화
- [ ] 수수료 모델 (약 1만원/건)
